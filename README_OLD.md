# カフェ在庫管理システム

CSVアップロードによるカフェ向け在庫管理システムです。

## 機能

### 在庫管理
- 📊 在庫統計の可視化（総商品数、在庫少、在庫切れ、総在庫価値）
- 📦 商品一覧の表示と編集（商品コード、名前、カテゴリ、価格、在庫数）
- 📁 CSVファイルでのインポート/エクスポート
- ⚠️ 発注アラート機能（在庫が最低レベル以下の商品を通知）

### 売上レポート (NEW!)
- 📈 商品別売上データの表示
- 💰 売上統計（総売上額、粗利総額、販売商品数）
- 🔤 Shift-JISエンコーディング対応のCSVインポート
- 📊 売上構成比の可視化
- 🔄 自動在庫連携（売上データから在庫を自動減算）

### 共通機能
- 💾 ローカルストレージでのデータ保存
- 🌓 ダークモード対応
- ✨ Linear風のモダンでミニマルなUI

## 技術スタック

- **フロントエンド**: SvelteKit 2 + Svelte 5 + TypeScript
- **ビルドツール**: Vite 5
- **スタイリング**: Tailwind CSS
- **デザインシステム**: Linear風 (Ultra-minimal, precise, purple accent)
- **アイコン**: Lucide Svelte
- **データ管理**: Svelte Stores + LocalStorage

## 使い方

### CSVファイルのインポート

1. ダッシュボードの「CSVアップロード」ボタンをクリック
2. CSVファイルを選択してアップロード
3. データが自動的にインポートされ、ローカルストレージに保存されます

#### CSVフォーマット

```csv
name,code,category,price,stockQuantity,minStockLevel,supplier
コーヒー豆 ブラジル,COFFEE-001,飲料,1200,45,20,コーヒー商社A
ミルク 1L,MILK-001,食品,250,18,30,地元乳業
```

必須列: `name`, `price`, `stockQuantity`

### CSVファイルのエクスポート

ダッシュボードの「CSVエクスポート」ボタンをクリックすると、現在の在庫データをCSV形式でダウンロードできます。

### 商品の編集

各商品の右側にある編集アイコンをクリックすると、商品情報を編集できます。変更は自動的にローカルストレージに保存されます。

## 売上レポート機能

### 売上CSVのインポート

1. 在庫管理ダッシュボードの「売上レポート」ボタンをクリック
2. 「売上CSVアップロード」ボタンをクリック
3. 商品別売上CSVファイルを選択してアップロード
4. **Shift-JISエンコーディングにも自動対応**
5. **自動在庫連携**: 売上データをインポートすると、販売商品数に基づいて在庫数が自動的に減算されます

### 自動在庫連携機能

売上CSVをインポートすると、システムが自動的に以下の処理を実行します：

1. **商品マッチング**: 以下の優先順位で在庫商品とマッチング
   - 商品コード（productCode）
   - 商品ID（productId）
   - 商品名（productName）
2. **在庫減算**: マッチした商品の在庫数から販売商品数を減算
3. **結果表示**: 更新された商品とマッチしなかった商品を表示

#### 売上CSVフォーマット

```csv
商品名,カテゴリー,税区分,販売総売上,構成比%,粗利総額,構成比%,販売商品数,構成比%,返品商品数,構成比%,商品ID,商品コード,バーコード
コーヒー,基本,内税,400,100.0,400,100.0,1,50.0,0,0.0,10000,,
```

必須列: `商品名`, `販売総売上`, `販売商品数`

### エンコーディング対応

- **Shift-JIS**: airレジなどから出力されるCSVに対応
- **UTF-8**: 標準的なCSVファイルに対応
- 自動判定により、どちらのエンコーディングでも正しく読み込めます
